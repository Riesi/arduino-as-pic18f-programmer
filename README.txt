INSTALLATION:

 - Debian/Ubuntu: 
   apt install arduino python-serial

HOW TO:

1. Upload the sketch to the arduino
2. Connect the chip to the arduino
3. Run ./pic_programmer.py -i HEX_FILE

* For help run ./pic_programmer.py -h

Thats it!
